# 日本語 Study
NHK Web Easy 기사를 기반으로 일본어를 학습하는 개인 학습 웹사이트

🔗 **[사이트 바로가기](https://jangdonguk.github.io/japanese-study/)**

## 기능
* 하루 한 줄 — 매일 바뀌는 일본어 문장 + 오늘의 단어 + 문법 포인트
* JLPT 연습 문제 — 매일 바뀌는 N2/N1 수준 문법·어휘·독해 문제 5개
* 한자 소나기 게임 — 하늘에서 떨어지는 한자를 보고 뜻을 타이핑하는 단어 암기 게임
* 속도 3단계 조절 (보통 / 빠름 / 매우빠름)
* 스트릭 & 레벨 시스템
* 노션 DB 자동 연동 — NHK 기사에서 수집한 단어를 노션에 추가하면 매일 자정 게임에 자동 반영
